# ncurses++

ncurses++ is an attempt to create high-level C++ interface for old and good ncurses library.

## Functionality

 - horizontal and vertical layouts
 - colors
 - static text
 - attributes

## Building

To build ncurses++ you need a compiler supporting C++17, cmake and ncurses.

1. mkdir build
2. cd build
3. cmake ..
4. make
